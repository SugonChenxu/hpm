## 1. 用户与认证管理（全部仅命令行）

> 后端 `auth.js` 仅暴露 `POST /api/auth/login`、`GET /api/auth/me`、`POST /api/auth/logout`，**无**增删改用户或改密码的接口；前端也无用户管理页。`auth.js` 注释提到由 `scripts/create-user` 写入，但当前 `scripts/` 下并不存在该脚本——目前需直接操作 `users` 表。
> `users` 表结构：`id, username(UNIQUE), password_hash, created_at`。

### 1.1 新增用户
```bash
cd /d/HPM/server
node -e "const bcrypt=require('bcryptjs'),db=require('better-sqlite3')('hpm.db');const h=bcrypt.hashSync('初始密码123',10);db.prepare('INSERT INTO users(username,password_hash) VALUES(?,?)').run('同事用户名',h);console.log('已创建用户：同事用户名');"
```
> 替换 `初始密码123` 与 `同事用户名`。密码经 bcrypt 哈希（与登录校验一致）。

### 1.2 修改 / 重置密码
```bash
cd /d/HPM/server
node -e "const bcrypt=require('bcryptjs'),db=require('better-sqlite3')('hpm.db');const h=bcrypt.hashSync('新密码',10);const r=db.prepare('UPDATE users SET password_hash=? WHERE username=?').run(h,'同事用户名');console.log('影响行数:',r.changes);"
```

### 1.3 查看用户列表
```bash
cd /d/HPM/server
node -e "const db=require('better-sqlite3')('hpm.db');console.table(db.prepare('SELECT id,username,created_at FROM users').all());"
```

### 1.4 删除账号（谨慎）
```bash
cd /d/HPM/server
node -e "const db=require('better-sqlite3')('hpm.db');const r=db.prepare('DELETE FROM users WHERE username=?').run('同事用户名');console.log('影响行数:',r.changes);"
```
> ⚠️ `users` 表无停用字段，删除为物理删除。业务表（`projects/tasks/issues/...`）通过 `owner_id` 关联，删用户后其历史数据变为“孤儿”（仍带原 owner_id，但查不到对应用户）。如仅想禁用登录，建议改密码或暂不改（后续可加 `is_active` 字段）。

## 2. 服务与进程管理（PM2 / start.sh）
```bash
bash start.sh status      # 查看状态
bash start.sh restart     # 重启
bash start.sh stop        # 停止
bash start.sh start       # 生产模式启动（单进程 3000）
bash start.sh build       # 编译前端并切生产模式
bash start.sh logs        # 查看日志

# 或直接 PM2
npx pm2 status
npx pm2 logs forge --lines 50
npx pm2 restart forge
```
> 单进程生产模式 `forge` 监听 3000；开发模式 `bash start.sh dev`（Vite 5173 + API 3001）。

## 3. 部署与网络（系统级，需管理员）

### 3.1 防火墙放行 3000 端口（让局域网同事访问）
```powershell
# 管理员 PowerShell
netsh advfirewall firewall add rule name="Forge-3000" dir=in action=allow protocol=TCP localport=3000
```
> 未放行则同事无法跨机访问本机 Forge。

### 3.2 确认本机局域网 IP（告知同事访问地址）
```bash
ipconfig | findstr IPv4
# 同事浏览器访问 http://<你的IP>:3000
```

### 3.3 开机自启（已配置，失效时重注册）
注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run\HPM-PM2-AutoStart` → `D:\HPM\scripts\pm2-resurrect.bat`。若失效：
```powershell
Set-ItemProperty "HKCU:\Software\Microsoft\Windows\CurrentVersion\Run" -Name HPM-PM2-AutoStart -Value "D:\HPM\scripts\pm2-resurrect.bat"
```
