# M4 — 物料管理模块详细 PRD

> **关联顶层 PRD**: `docs/PRD.md` v2.0  
> **开发优先级**: ④  
> **依赖**: M1 项目进度模块  
> **实现状态**: ✅ 已完成（2026-07-16）

---

## 一、模块核心定位

实现硬件项目物料全生命周期管理，支持 **Excel 批量导入 + 手动单行添加** 双模式数据录入，覆盖物料全字段编辑、状态流转、批量操作、导入导出、搜索过滤全流程，所有操作实时持久化存储。

---

## 二、表格基础规范

1. **固定列顺序**（从左至右）：序号、物料号、厂家、物料型号、物料状态、数量、采购时间、采购周期、预计交期、备注；
2. **全表所有列**均支持升序 / 降序排序，点击列头切换排序方向，同步展示排序状态可视化标识（▲/▼）；
3. **所有列支持手动拖拽调整列宽**，列宽配置持久化（localStorage）生效。

---

## 三、Excel 批量导入功能规范

1. 入口：工具栏「导入 Excel」按钮，仅支持 `.xlsx` / `.xls` 文件上传；
2. **列名智能识别**：自动识别列名，按预设映射规则匹配系统字段，兼容中英文常见列名；
3. 预览确认：解析完成后弹窗展示前 50 行数据预览，支持核对字段映射结果，确认后触发批量写入；
4. 数据写入：批量写入时自动为新增物料分配连续序号，与现有表格序号无缝衔接；
5. 异常处理：解析失败 / 列名无匹配 / 数据格式错误时展示友好报错，精准定位错误行与原因。

### 列名映射规则

| Excel 列名（兼容值） | 系统字段 |
| --- | --- |
| 序号 / seq / no | seq（忽略，由后端重新分配连续序号） |
| 物料状态 / 状态 / status | material_status |
| 厂家 / 供应商 / 品牌 / manufacturer | manufacturer |
| 物料型号 / 型号 / 规格 / model | model |
| 物料号 / 料号 / 编号 / partNumber | part_number |
| 数量 / quantity / qty | quantity |
| 单套用量 / 单套数量 | quantity_per_set |
| 套数 / 总套数 | set_count |
| 采购时间 / 采购日期 | purchase_date |
| 采购周期 / 周期 | lead_time |
| 预计交期 / 交期 / expectedDelivery / delivery | expected_delivery |
| 备注 / 说明 | notes |

> 注：规范映射表原仅列前 10 项，实现补齐 `数量` 与 `预计交期` 映射，以覆盖固定表格全字段。

---

## 四、手动添加功能规范

1. 入口：工具栏「添加一行」按钮，点击后在表格末尾插入新行；
2. 默认值：新行 `material_status` 默认填充为「默认」；
3. 交互优化：新行插入后自动聚焦「物料号」列编辑单元格，可直接输入；支持连续快速添加多行。

---

## 五、物料状态列交互规范

1. 渲染样式：状态列渲染为紧凑彩色标签，标签右侧带 ▾ 小箭头提示可交互；
2. 状态枚举与样式：

| 状态 | 文字颜色 | 背景色 | 业务说明 |
| --- | --- | --- | --- |
| 默认 | #8c8c8c | #f5f5f5（灰） | 新物料初始状态 |
| 已入库 | #fff | #52c41a（绿） | 物料已到货入库 |
| 已下单 | #135200 | #b7eb8f（浅绿） | 已下采购单待到货 |
| 待决策 | #fff | #faad14（黄） | 需内部决策 / 评审待确认 |
| 高风险 | #fff | #ff4d4f（红） | 存在延期 / 断供等风险 |

3. 交互逻辑：点击状态标签弹 Dropdown 菜单，5 个状态选项均带彩色预览，选中后标签样式与数据立即更新，无需额外保存。

---

## 六、全表格内联编辑规范

1. 触发方式：可编辑单元格点击即进入编辑模式，编辑态高亮；
2. 保存 / 取消：失焦或回车自动保存；Esc 取消本次编辑；
3. 键盘导航：Tab 横向下一个可编辑单元格，Shift+Tab 回跳；
4. 字段适配：文本类=输入框；数字类（数量 / 采购周期）= 数字输入；日期类（采购时间 / 预计交期）= DatePicker（中文月份、快捷选今日）。

---

## 七、行选择与批量操作规范

1. 行选择：首列 checkbox 多选，支持单行 / 全选 / 清除；
2. 批量操作栏：选中任意行后顶部显示蓝色工具栏，支持批量改状态 / 批量删除 / 批量导出；
3. 操作确认：批量删除弹二次确认弹窗，确认后执行。

---

## 八、撤销导入功能规范

1. 触发：每次 Excel 批量导入成功后，工具栏临时显示「撤销导入」按钮（带 5 分钟倒计时）；
2. 逻辑：点击移除最近一次导入的全部物料数据，恢复至导入前状态；
3. 有效期：仅在最近一次导入后 5 分钟内可见，页面刷新后仍可用（快照持久化至库）；
4. 快照覆盖：二次导入覆盖上一次撤销快照，仅支持撤销最近一次导入。

---

## 九、导出 Excel 功能规范

1. 入口：工具栏「导出 Excel」（全量）/ 批量栏「导出选中」；
2. 内容：序号、物料号、厂家、物料型号、物料状态、数量、采购时间、采购周期、预计交期、备注全字段，保留状态的文本与颜色标识（exceljs 填充）。

---

## 十、搜索过滤功能规范

1. 搜索入口：表格顶部搜索框，多维度模糊搜索；
2. 范围：物料号 / 厂家 / 物料型号 / 物料状态 / 备注，实时过滤，无匹配时展示空状态提示。

---

## 十一、全局开发约束

1. 所有操作完成后固定当前滚动位置，不回弹至页面顶部；
2. 完成后推送代码至 GitHub，生成版本变更日志，更新本模块 PRD；
3. 所有写入做合法性校验（数值容错、日期解析），异常展示友好提示，避免脏数据；
4. 序号列保持自上而下连续顺排，新增 / 删除后自动全局重排，无断档、无重复。

---

## 十二、数据模型

### `materials`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | INTEGER PK | — |
| project_id | INTEGER FK | 所属项目 |
| seq | INTEGER | 连续序号（增删后重排） |
| part_number | TEXT | 物料号 |
| manufacturer | TEXT | 厂家 |
| model | TEXT | 物料型号 |
| material_status | TEXT | 默认 / 已入库 / 已下单 / 待决策 / 高风险 |
| quantity | REAL | 数量 |
| quantity_per_set | REAL | 单套用量 |
| set_count | INTEGER | 套数 |
| purchase_date | TEXT | 采购时间 |
| lead_time | INTEGER | 采购周期（天） |
| expected_delivery | TEXT | 预计交期 |
| notes | TEXT | 备注 |
| created_at / updated_at | TEXT | 时间戳 |

### `material_import_snapshots`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | INTEGER PK | — |
| project_id | INTEGER | 所属项目 |
| ids_json | TEXT | 本次导入的物料 id 数组（JSON） |
| created_at | TEXT | 导入时间（用于 5 分钟窗口校验） |

---

## 十三、接口

| Method | Path | 说明 |
| --- | --- | --- |
| GET | /api/materials?project_id=&status=&search= | 列表（搜索 / 状态过滤） |
| POST | /api/materials | 单行新增（material_status 默认「默认」，自动分配序号） |
| GET | /api/materials/:id | 物料详情 |
| PUT | /api/materials/:id | 更新字段（序号 / 项目不可变） |
| DELETE | /api/materials/:id | 删除（重排序号） |
| POST | /api/materials/batch | 批量导入（分配连续序号 + 记录撤销快照） |
| DELETE | /api/materials/batch | 批量删除（重排序号） |
| PUT | /api/materials/batch-status | 批量修改状态 |
| GET | /api/materials/import-snapshot?project_id= | 最近一次导入快照 |
| POST | /api/materials/import-undo | 撤销最近一次导入（5 分钟窗口） |
