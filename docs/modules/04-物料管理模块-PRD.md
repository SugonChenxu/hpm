## 一、模块职责

**统一记录、维护各类物料信息**。涵盖通用物料和开发物料两大类，以项目为单位管理物料清单（BOM），跟踪每项物料的备料交期状态，实现逾期自动预警。


【物料管理】需求
硬件项目的物料清单管理。以 **Excel 导入为主**，手动添加为辅。所有字段均可像 Excel 一样点击内联编辑。

## 二、功能清单

### 2.1 物料清单管理

| 功能 | 描述 |
|------|------|
| **物料列表** | 表格视图——料号/名称/规格/类型(通用/开发)/数量/供应商/交期/状态 |
| **创建物料** | 单条创建：料号(必填)、名称(必填)、规格、类型、数量、供应商、计划交期、备料周期(天)、所属项目(必选) |
| **批量导入** | 支持 CSV/Excel 粘贴导入：料号,名称,规格,类型,数量,供应商,计划交期,备料周期——系统校验后批量写入 |
| **编辑/删除** | 逐条编辑或软删除 |
| **物料详情** | 侧边栏：完整字段 + 交期时间线（计划→实际到货）+ Second Source 列表 |

### 2.2 物料类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **通用物料** | 通用部件，带正式料号，多项目共享 | CPU/内存/硬盘/网卡/风扇/电源模块 |
| **开发物料** | 项目专用定制部件 | 自研主板/背板/Riser卡/定制线缆/定制散热器 |
| **包材** | 包装材料 | 纸箱/泡棉/标签/说明书 |

### 2.3 备料跟踪

| 功能 | 描述 |
|------|------|
| **交期状态** | 每项物料状态：待下单 → 已下单 → 在途 → 已到货 |
| **交期倒计时** | 以计划交期为基准，前端实时显示倒计时天数 |
| **逾期预警** | 超过计划交期且未到货——状态自动标红 + 仪表盘徽标计数 |
| **临期提醒** | 距计划交期 ≤ 3 天——状态黄色 |
| **实际到货记录** | 到货后填写实际到货日期和到货数量，系统自动计算交期偏差（days_late） |
| **备料周期参考** | 创建物料时根据知识库预设值给出周期建议：主板 2 月 / 小板卡 1 月 / PCB mid loss 2-3 周 / 结构 4-6 周 / 散热 4-6 周 / 包材 3 天 |

### 2.4 物料视图与筛选

| 功能 | 描述 |
|------|------|
| **按项目筛选** | 全部项目 / 指定项目 |
| **按类型筛选** | 通用 / 开发 / 包材 |
| **按状态筛选** | 待下单 / 已下单 / 在途 / 已到货 / 已逾期 |
| **搜索** | 按料号、名称、供应商模糊搜索 |
| **备料看板** | 可选看板视图——按交期状态分列，拖拽物料卡片更新状态 |

## 三、数据模型

### 3.1 `materials`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | — |
| project_id | INTEGER FK | 所属项目 |
| part_no | TEXT | 料号（必填） |
| name | TEXT | 物料名称（必填） |
| spec | TEXT | 规格描述 |
| material_type | TEXT | 通用 / 开发 / 包材 |
| quantity | INTEGER | 数量 |
| supplier | TEXT | 供应商 |
| lead_time_days | INTEGER | 备料周期（天） |
| planned_delivery | TEXT | 计划交期 ISO8601 |
| actual_delivery | TEXT | 实际到货日期（可空） |
| actual_quantity | INTEGER | 实际到货数量 |
| status | TEXT | 待下单/已下单/在途/已到货/已逾期 |
| is_second_source | INTEGER | 是否备选物料(Second Source)，0/1 |
| parent_material_id | INTEGER FK | 关联主物料（Second Source 指向主料） |
| notes | TEXT | 备注 |
| created_at | TEXT | — |
| updated_at | TEXT | — |

## 四、接口

| Method | Path | 说明 |
|--------|------|------|
| GET | /api/materials | 物料列表（?project_id=&type=&status=&search=） |
| POST | /api/materials | 创建物料 |
| POST | /api/materials/batch | 批量导入（JSON 数组） |
| GET | /api/materials/:id | 物料详情 |
| PUT | /api/materials/:id | 更新物料（含到货确认） |
| DELETE | /api/materials/:id | 软删除 |
| GET | /api/materials/overdue | 逾期物料列表 |
| GET | /api/materials/stats | 物料统计（按项目：总数/逾期数/在途数） |
