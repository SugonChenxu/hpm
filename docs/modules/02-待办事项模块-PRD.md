# M2 — 待办事项模块详细 PRD

> **关联顶层 PRD**: `docs/PRD.md` v2.0  
> **开发优先级**: ②  
> **依赖**: M1 项目进度模块（待办需关联项目/阶段）

---

## 一、模块职责

**待办事项可视化看板管理**。以 Kanban（看板）为主视图，支持任务在不同状态列之间拖拽流转，按项目/阶段/优先级/截止日期多维度筛选。任务可关联到 M1 模块中的项目与阶段。

---

## 二、功能清单

### 2.1 看板视图

| 功能 | 描述 |
|------|------|
| **看板列** | 默认四列：待开始 → 进行中 → 待验证 → 已完成。用户可自定义增删列、改名、调序 |
| **任务卡片** | 每张卡片显示：标题、优先级色标(P0红/P1橙/P2蓝)、截止日期、负责人、所属项目标签 |
| **拖拽流转** | 拖拽卡片到目标列即更新任务状态，无需弹窗确认 |
| **快速创建** | 每列底部「+ 添加任务」按钮，输入标题+回车即可创建（其他字段默认空，可后续编辑） |

### 2.2 任务管理

| 功能 | 描述 |
|------|------|
| **任务详情** | 点击卡片展开侧边栏/弹窗：标题、描述、优先级(P0/P1/P2)、负责人、截止日期、关联项目/阶段 |
| **创建/编辑** | 完整表单：标题(必填)+描述+优先级+负责人+截止日期(datetime picker)+关联项目(下拉选择)+关联阶段(根据项目联动) |
| **删除** | 软删除，已删除任务在垃圾箱中保留 30 天可恢复 |
| **批量操作** | 多选任务卡片 → 批量修改优先级/负责人/状态/截止日期 |

### 2.3 多视图筛选

| 功能 | 描述 |
|------|------|
| **按项目筛选** | 顶部下拉选择器：全部项目 / 指定项目 |
| **按阶段筛选** | 在选择项目后联动：全部阶段 / 指定阶段 |
| **按优先级筛选** | 快速切换：全部 / P0 / P1 / P2 |
| **按负责人筛选** | 下拉选择器 |
| **截止日期视图** | 切换至列表视图，按截止日期升序排列，逾期标红 |
| **我的待办视图** | 全局过滤为「负责人=我」的任务，跨项目聚合 |

### 2.4 截止日期追踪

| 功能 | 描述 |
|------|------|
| **逾期标记** | 任务卡片右下角显示距截止日期的天数——逾期红色负数、今天黄色「今天」、3 天内橙色、3 天以上灰色 |
| **仪表盘徽标** | 全局仪表盘顶部显示「逾期待办: N 项」红色徽标，点击跳转至逾期任务列表 |

## 三、数据模型

### 3.1 `tasks`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | — |
| project_id | INTEGER FK | 关联项目（可选，支持全局待办） |
| phase_id | INTEGER FK | 关联阶段（可选） |
| title | TEXT | 任务标题 |
| description | TEXT | 详细描述（Markdown） |
| priority | TEXT | P0 / P1 / P2 |
| assignee | TEXT | 负责人（字符串，非用户表引用） |
| kanban_column | TEXT | 所属看板列名 |
| due_date | TEXT | 截止日期 ISO8601 |
| status | TEXT | 与 kanban_column 联动（可通过映射表） |
| created_at | TEXT | — |
| updated_at | TEXT | — |
| deleted_at | TEXT | 软删除时间戳（NULL=未删除） |

### 3.2 `kanban_columns`（看板列配置，用户级）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | — |
| name | TEXT | 列名（如「进行中」） |
| column_order | INTEGER | 列序 |
| color | TEXT | 列顶部色条颜色 |
| is_default | INTEGER | 是否预设列（不可删） |

## 四、接口

| Method | Path | 说明 |
|--------|------|------|
| GET | /api/tasks | 任务列表（?project_id=&phase_id=&priority=&assignee=&status=） |
| POST | /api/tasks | 创建任务 |
| GET | /api/tasks/:id | 任务详情 |
| PUT | /api/tasks/:id | 更新任务（含拖拽后的 kanban_column 变更） |
| DELETE | /api/tasks/:id | 软删除 |
| PUT | /api/tasks/batch | 批量更新 |
| GET | /api/tasks/overdue | 逾期任务列表 |
| GET | /api/kanban-columns | 看板列配置 |
| PUT | /api/kanban-columns | 更新列配置 |
