### BugFix: Vite --force 无限循环 + fastRefresh:false 导致白屏
- **根因1**: `ecosystem.config.js` 的 `--force` 标志导致 Vite 每次启动强制重新优化依赖 → 优化中崩溃 → PM2 重启 → 又 --force → 死循环
- **根因2**: `@vitejs/plugin-react` v6 设 `fastRefresh:false` 后仍注入 `$RefreshSig$()` 但不加载 Refresh 运行时 → ReferenceError → 白屏
- **修复**: 移除 `--force` 标志 + `NODE_ENV` 改为 development + 移除 `fastRefresh:false`（恢复默认 Fast Refresh，HTML 正确注入 /@react-refresh preamble）
