## M8 用户管理模块（已实装部署 + 推送 commit 52d7f86）
- 用户采纳建议：把「加用户 / 改密码」做成网页用户管理页。
- **后端** `server/src/routes/users.js`：GET 列表 / POST 新增 / POST me/password 改自己 / PUT :id/password 重置他人 / DELETE :id 删除（禁删自己）；bcrypt(cost10)；受 requireAuth 保护。
- **关键 bug**：初版误挂 `app.use("/api", usersRouter)` → router 内 `get("/")` 匹配 `/api/` 而非 `/api/users`，返回 404。修正为 `app.use("/api/users", usersRouter)`。（经验：Express 子路由挂载点要含完整前缀路径）
- **前端**：`pages/UserManagementPage.jsx`（列表/新增/重置/删除）+ `components/common/ChangePasswordDialog.jsx`（改自己密码，校验原密码）+ `api.users` 封装；App `/users` 路由 + Sidebar 菜单项（内联 SvgIcon，遵循 Vite8 禁 @mui/icons-material）+ Layout AppBar「修改密码」入口。
- 端到端验证全通过（登录/列表/新增/改密/重置/禁删自己400/删他人）。已 build 部署到 PM2 forge（3000），前端 /users 返回 200。
