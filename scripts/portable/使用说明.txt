============================================================
Forge 硬件项目管理工具 —— 使用说明
============================================================

【这是什么】
Forge 是一款面向硬件产品研发全流程的项目管理工具（项目进度 / 待办 /
故障 / 物料 / 会议计划 / 周报）。本包为「绿色版」，已内置运行环境，
无需安装 Node.js、无需联网构建，解压即可用。

------------------------------------------------------------
一、首次启动（最关键一步）
------------------------------------------------------------
1. 将整个 Forge 文件夹解压到任意位置（如 D:\Forge、桌面均可）。
2. 双击「启动.bat」。
3. 弹出的黑色窗口会显示两行访问地址：
      本机访问:  http://localhost:3000
      局域网地址: 192.168.x.x:3000   （同事访问用，见第四节）
4. 打开浏览器访问 http://localhost:3000 即可使用。
5. 关闭窗口即停止服务（Ctrl+C 也可）。

提示：首次启动会自动在 Forge\server\ 下生成数据库文件 hpm.db，
      之后所有数据都保存在这里。

------------------------------------------------------------
二、数据备份
------------------------------------------------------------
- 双击「备份.bat」：会在 Forge\backups\ 下生成带时间戳的
  hpm.db 副本，建议定期备份。
- 迁移电脑：直接把整个 Forge 文件夹拷走即可，数据随文件夹走。
- 重要：不要手动删除 server\hpm.db，否则数据丢失。

------------------------------------------------------------
三、Mantis 缺陷同步（必需配置）
------------------------------------------------------------
Forge 可双向同步 Mantis 缺陷数据。每人需配置自己的 Mantis 账号：
1. 浏览器打开 Forge → 进入「故障管理」模块。
2. 找到 Mantis 设置入口（通常为页面内的“设置 / 连接配置”）。
3. 填写：Mantis 服务器地址（如 https://mantis.xxx.com）、
   你的 API Token（在 Mantis 个人设置页生成）。
4. 点「测试连接」，提示成功即配置完成。
5. 之后即可在故障管理页「同步」拉取缺陷。

注意：Mantis Token 是个人凭证，请在自己电脑上配置，不要外传。

------------------------------------------------------------
四、让同事通过局域网访问（可选）
------------------------------------------------------------
场景：你在一台电脑上当“主机”，同事用各自浏览器访问你的地址。
1. 主机双击「启动.bat」正常启动。
2. 主机双击「防火墙放行.bat」（需以管理员身份运行，否则可能被拦截）。
3. 同事浏览器打开启动窗口里显示的「局域网地址」即可。
   例如：http://192.168.1.50:3000

⚠️ 安全提醒：本工具为内部便捷版，没有登录密码。局域网共享模式下，
   同网段任何人都能读写全部数据。仅建议在互相信任的内网使用，
   且不要让陌生人连入同一网络。

------------------------------------------------------------
五、OA 物料导入（选装）
------------------------------------------------------------
如需从 OA 系统一键导入物料，请使用配套的浏览器扩展 / 书签脚本
（由管理员另行提供）。扩展会向本机 http://localhost:3000 注入数据，
无需额外配置。

------------------------------------------------------------
六、升级
------------------------------------------------------------
收到新版本压缩包后：
1. 先双击「备份.bat」备份数据。
2. 用新包覆盖旧包的 server\ 与 client\ 目录（不要覆盖 server\hpm.db）。
3. 重新双击「启动.bat」。
或直接：删除旧包、解压新包，再把旧包的 server\hpm.db 复制到新包同名位置。

------------------------------------------------------------
七、常见问题
------------------------------------------------------------
Q: 双击启动.bat 闪退 / 无法访问？
A: 确认 3000 端口未被其他程序占用；或改端口：右键启动.bat 编辑，
   把 set PORT=3000 改成其他端口（如 8080），同事访问时也用新端口。

Q: 提示“端口已被占用”？
A: 关闭占用 3000 端口的程序，或修改上面的 PORT 值。

Q: 公司电脑限制运行 .bat？
A: 联系 IT 将 Forge 文件夹加入白名单；或右键“以管理员身份运行”。

Q: 想开机自动启动？
A: 双击「开机自启.bat」即可（会在启动文件夹放一个快捷方式）。
